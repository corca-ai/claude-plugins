# I1-R60 `cwf:run` end-to-end 실행

## 목적

tiny task로 `cwf:run` 전체 체인을 가능한 범위까지 실제 진행하고 hard stop 지점을 확인한다.

## 실행

- 로그: [project/iter1/artifacts/I1-R60.log](../artifacts/I1-R60.log)
- 명령: `claude --print ... "cwf:run ..."`

## 관찰

- Claude 경로 90초 timeout(`CLAUDE_EXIT=124`), Gemini도 timeout
- 동일 시간대 run 관련 아티팩트는 일부 생성됨
  - 예: [run project 02](../sandbox/user-repo-b/.cwf/projects/260219-02-hello-script-run), [run project 03](../sandbox/user-repo-b/.cwf/projects/260219-03-hello-script-run)
- 그러나 `run-stage-provenance.md`에 stage 기록이 비어 있어 정확한 hard stop phase를 자동 산출하지 못함

## 판정

- 상태: FAIL(timeout)
- 결론: non-interactive 단일 호출로는 end-to-end 완료/phase 보고가 안정적으로 닫히지 않음
