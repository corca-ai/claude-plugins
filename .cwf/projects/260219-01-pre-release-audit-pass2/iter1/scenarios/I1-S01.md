# I1-S01 marketplace 추가 + 플러그인 설치(project scope)

## 목적

별도 레포의 신규 유저가 README 안내대로 marketplace 추가 후 `cwf`를 설치할 수 있는지 확인한다.

## 실행 일시

2026-02-19

## 실행 환경

- 샌드박스 레포: [.cwf/projects/260219-01-pre-release-audit-pass2/iter1/sandbox/user-repo-a](../sandbox/user-repo-a)
- 실행 로그: [.cwf/projects/260219-01-pre-release-audit-pass2/iter1/artifacts/I1-S01.log](../artifacts/I1-S01.log)

## 실행 명령

```bash
claude plugin marketplace add https://github.com/corca-ai/claude-plugins.git
claude plugin install cwf@corca-plugins --scope project
claude plugin list
```

## 관찰 결과

1. marketplace 추가는 성공함
   - `✔ Successfully added marketplace: corca-plugins`
2. `cwf@corca-plugins` 설치는 실패함
   - `✘ Failed to install plugin "cwf@corca-plugins": Plugin "cwf" not found in marketplace "corca-plugins"`
3. 설치 목록에는 `cwf`가 나타나지 않음

## 판정

- 상태: FAIL (의도 불일치, 경로 중단)
- 심각도: Blocker
- 영향: README의 기본 설치 경로(`cwf@corca-plugins`)를 신규 유저가 그대로 수행할 수 없음

## 즉시 조치

- 이 경로는 중단
- 마스터 시나리오에 blocker로 반영
- 설치 우회 경로(로컬 플러그인 디렉터리/manifest 확인)로 나머지 setup/hook/skill 검증을 계속 진행
