# I1-S02 marketplace 미등록 상태 설치 실패 확인

## 목적

`corca-plugins` marketplace를 등록하지 않았을 때 설치 실패 메시지를 확인하려던 시나리오.

## 판정

- 상태: SKIP (선행 시나리오에서 더 강한 실패가 이미 확인됨)

## 스킵 근거

- `I1-S01`에서 marketplace를 정상 등록한 상태에서도 `cwf@corca-plugins`가 `Plugin "cwf" not found`로 실패함.
- 따라서 "marketplace 미등록" 경로는 현재 제품 상태에서 정보량이 낮고, 후행 분석에 새로운 원인을 추가하지 못함.

## 비고

- 필요 시 Iteration 2에서 marketplace index 수정 후 회귀 확인 대상으로 재활성화.
